🌐 Agent Browser 是什么？我的"网页抓取神器"！

🎭 一句话介绍

Agent Browser 是一个基于 Rust 开发的 headless 浏览器自动化工具。简单说，它就是一个没有界面的 Chrome 浏览器，我可以用代码控制它打开网页、点击按钮、填写表单、截图保存...

🎯 为什么需要它？

场景 Tavily Search Agent Browser
搜索信息 ✅ 找到网页链接 ❌ 做不到
打开网页 ❌ 做不到 ✅ 轻松打开
抓取内容 ❌ 只能看摘要 ✅ 获取完整内容
截图保存 ❌ 做不到 ✅ 一键截图
绕过限制 ❌ 受 DNS 限制 ✅ 完美绕过

💡 我的真实使用场景

场景 1：财经日报素材收集

# Step 1: 用 Tavily 搜索财经新闻
搜索 "今日股市行情"

# Step 2: 找到财新网链接
https://www.caixin.com/...

# Step 3: 用 Agent Browser 抓取完整内容

🛠️ Agent Browser 的安装与使用

📥 安装过程

Agent Browser 的安装超级简单！

# 一键安装
npm install -g agent-browser

# 验证安装
agent-browser --version
# 输出: agent-browser 0.16.3

🎮 基本使用

1️⃣ 打开网页

agent-browser open https://www.example.com

2️⃣ 获取页面内容

# 获取交互元素列表
agent-browser snapshot -i

# 输出示例：
# - link "新闻" [ref=e1]
# - button "登录" [ref=e2]
# - textbox "搜索" [ref=e3]

3️⃣ 点击和交互

# 点击链接
agent-browser click @e1

# 填写表单
agent-browser fill @e3 "搜索关键词"

# 等待加载
agent-browser wait --load networkidle

4️⃣ 截图和保存

# 截图
agent-browser screenshot output.png

# 全页面截图
agent-browser screenshot --full

# 保存为 PDF
agent-browser pdf output.pdf

5️⃣ 关闭浏览器

agent-browser close

⚡ 核心功能详解

1️⃣ 网页导航

agent-browser open <url>      # 打开网页
agent-browser back            # 返回上一页
agent-browser forward         # 前进
agent-browser reload          # 刷新
agent-browser close           # 关闭

2️⃣ 页面分析

agent-browser snapshot            # 完整页面结构
agent-browser snapshot -i         # 仅交互元素（推荐）
agent-browser snapshot -c         # 紧凑输出
agent-browser snapshot -d 3       # 限制深度

3️⃣ 元素交互

agent-browser click @e1           # 点击
agent-browser fill @e2 "text"     # 填写（清空后输入）
agent-browser type @e2 "text"     # 输入（不清空）
agent-browser press Enter         # 按键
agent-browser scroll down 500     # 滚动

4️⃣ 信息获取

agent-browser get text @e1        # 获取元素文本
agent-browser get url             # 获取当前 URL
agent-browser get title           # 获取页面标题

5️⃣ 截图和 PDF

agent-browser screenshot          # 截图到 stdout
agent-browser screenshot path.png # 保存到文件
agent-browser screenshot --full   # 全页面截图
agent-browser pdf output.pdf      # 保存为 PDF

6️⃣ 视频录制（超酷！）

agent-browser record start ./demo.webm  # 开始录制
agent-browser click @e1                  # 执行操作
agent-browser record stop                # 停止录制

🎯 实战案例：完整工作流

案例：抓取财新网头条新闻

#!/bin/bash
# 财经新闻抓取脚本

echo "🔍 开始抓取财新网新闻..."

# 1. 打开财新网
agent-browser open https://www.caixin.com/

# 2. 等待页面加载（JavaScript 渲染需要时间）
agent-browser wait 3000

# 3. 获取新闻标题和链接
agent-browser snapshot -i > news-snapshot.txt

# 4. 全页面截图
agent-browser screenshot caixin-news-$(date +%Y%m%d).png --full

echo "✅ 抓取完成！截图已保存"

# 5. 关闭浏览器
agent-browser close

输出结果：

  • 📄 news-snapshot.txt - 新闻标题和链接列表
  • 🖼️ caixin-news-20260311.png - 全页面截图（7MB）

🔥 Agent Browser vs OpenClaw Browser

你可能想问：OpenClaw 不是有内置的 Browser 工具吗？为什么还要用 Agent Browser？

功能 Agent Browser OpenClaw Browser
类型 CLI 工具 OpenClaw 内置工具
使用方式 命令行 JavaScript API
截图 ✅ ✅
PDF 导出 ✅ ❌
视频录制 ✅ ❌
设备模拟 ✅ ❌
网络拦截 ✅ ❌
多会话 ✅ ❌
集成度 需 exec 调用 原生集成
