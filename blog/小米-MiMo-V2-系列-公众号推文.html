小米发布 MiMo V2 系列大模型：全面布局 Agent 时代 🔬

3 月 19 日凌晨，小米正式推出面向 Agent 时代的三款自研大模型

2026 年 3 月 19 日凌晨，小米公司发布了 MiMo V2 系列大模型。这个系列包括三款产品：MiMo-V2-Pro、MiMo-V2-Omni 和 MiMo-V2-TTS，分别面向不同的应用场景和技术方向。

目前，这三款模型已上线 Xiaomi MiMo、MiMo Studio、金山办公、小米浏览器等平台，并通过 OpenClaw、OpenCode、KiloCode、Blackbox、Cline 等开发框架接入使用。

MiMo-V2-Pro：万亿参数的 Agent 旗舰

作为本次发布的核心产品，MiMo-V2-Pro 拥有超过 1T 的总参数量（42B 激活参数），采用创新的混合注意力架构，支持 1M Token 超长上下文。

在权威大模型综合智能排行榜 Artificial Analysis 上，该模型位列全球第八、国内第二。这一排名使其与 Claude Sonnet 4.6、GPT 5.2、Gemini 3.0 Pro 处于同一梯队。

从技术报告披露的数据来看，MiMo-V2-Pro 在多个评测基准上表现突出：

评测基准 表现
GDPval-AA Elo 1426，超越 GLM-5 推理版、Kimi K2.5 推理版
AA-Omniscience 得分 +5，幻觉率控制较好
PinchBench / ClawEval 效果处于全球顶尖水平

值得一提的是，一周前代号为 "Hunter Alpha" 的匿名模型曾在 OpenRouter 平台引发关注，多天登顶日榜，调用量突破 1T tokens。小米官方确认，这正是 MiMo-V2-Pro 的早期测试版本。

在应用场景方面，MiMo-V2-Pro 针对复杂多样的 Agent Scaffold 进行了 SFT 和 RL 训练。官方表示，在 OpenClaw、Claude Code 等智能体框架中，该模型能够在无人工干预的条件下完成复杂工作流编排、长程规划与精准工具调用。

定价策略上，MiMo-V2-Pro 采取分段计价：≤ 256K 上下文时，输入 $1.00、输出 $3.00 / 百万 tokens；> 256K 时，输入 $2.00、输出 $6.00 / 百万 tokens。这一定价约为 Claude Sonnet 4.6 的五分之一。

MiMo-V2-Omni：全模态的感知与执行

如果说 Pro 专注于文本理解与推理，那么 Omni 则实现了看得见、听得懂、能动手的全模态能力。

该模型支持音频、图像和视频理解：

• 音频理解：支持超过 10 小时连续长音频的深度理解
• 图像理解：具备多学科视觉推理与复杂图表分析能力，官方称超越 Claude Opus 4.6
• 视频理解：原生音视频联合输入，具备情境感知与未来推理能力

MiMo-V2-Omni 采用轻量高效设计，支持 256K Token 上下文。官方表示，该模型能够跨模态理解复杂环境、自主制定并执行计划、在遇到异常时实时修正策略。

目前，MiMo-V2-Omni 已正式开放 API 服务，输入 $0.4、输出 $2 / 百万 tokens。同时，该模型联合 OpenClaw、OpenCode、KiloCode、Blackbox 及 Cline 五大 Agent 开发框架，为全球开发者提供为期一周的限时免费接口。

MiMo-V2-TTS：语音合成的新进展

除了文本和多模态模型，小米还发布了自主研发的语音合成大模型 MiMo-V2-TTS。

该模型基于自研 Audio Tokenizer 和多码本语音-文本联合建模架构，经过上亿小时语音数据的预训练与多维度强化学习，实现了多粒度语音风格控制。

具体能力包括：

✅ 整体到局部的风格控制：通过自然语言指令设定整体语音基调，同时对句内局部片段进行细粒度情绪调节
✅ 自然韵律还原：在同一句话内完成语气转折和情感递变
✅ 歌声合成：准确表达音高和节奏
✅ 多方言支持：东北话、四川话、河南话、粤语、中国台湾腔等

小米表示，MiMo-V2-TTS 是其语音技术路线图上的重要里程碑。未来规划包括更多语种覆盖，以及与 MiMo-V2-Omni 多模态理解能力的深度融合。

生态整合：从模型到应用

MiMo V2 系列的发布不仅是模型层面的更新，更涉及生态层面的整合。

在办公场景，MiMo V2 已与金山办公生态实现框架级集成。WPS 灵犀现已接入 MiMo-V2-Pro 模型，原生支持 Word、Excel、PPT、PDF 四大主流格式。

在开发工具方面，MiMo-V2-Pro 和 MiMo-V2-Omni 已支持 OpenClaw、OpenCode、KiloCode、Blackbox、Cline 等框架接入，开发者可限时免费体验。

此外，小米此前发布的 MiMo-V2-Flash（总参数 309B，激活 15B）已在 HuggingFace 开源（MIT 协议）。该模型在代码和 Agent 评测中稳居全球开源 Top 2，API 定价为输入 0.7 元、输出 2.1 元 / 百万 tokens。

市场定位与行业观察

从产品线布局来看，小米构建了覆盖不同参数规模和能力方向的模型矩阵：Flash（开源轻量）、Pro（旗舰文本）、Omni（全模态）、TTS（语音合成）。

这一定位与 OpenAI、Google、Anthropic 等国际厂商的技术路线基本一致，但在定价策略上体现了差异化竞争思路。MiMo-V2-Pro 以万亿参数规模进入全球第一梯队，定价却仅为同类产品的五分之一；MiMo-V2-Flash 则完全开源，降低使用门槛。

值得关注的是，小米选择开源 + 商业 API的双轨策略：Flash 完全开源满足研究社区需求，Pro、Omni、TTS 通过 API 服务实现商业化。这一策略与业界主流做法相符，既保持技术开放性，也为长期发展留出空间。

对于开发者和企业用户而言，MiMo V2 系列提供了新的选择，特别是在 Agent 框架集成、办公场景应用等方向具有实际落地价值。
