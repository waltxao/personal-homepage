先了解ClawHub：OpenClaw的"App Store"

ClawHub是什么？

ClawHub是OpenClaw的公共技能注册表，你可以把它理解为OpenClaw的App Store。所有技能都是公开、免费的，任何人都可以搜索、安装和发布。

关键数据一览：

指标 数据
总注册技能数 24,759+
筛选后优质技能 5,000+（通过安全检测）
分类数 32个主要类别

安装方式：

# 安装ClawHub CLI（如果还没装）
npm i -g clawhub

# 安装技能
clawhub install <skill-slug>

# 查看已安装
clawhub list
